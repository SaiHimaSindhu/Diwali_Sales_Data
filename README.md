# 🪔 Diwali Sales Data Analysis

## 📌 Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on the Diwali Sales dataset using Python, Pandas, Matplotlib, and Seaborn. The main objective of this project is to analyze customer purchasing behavior during Diwali sales and generate meaningful business insights.

---

# 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

# 📂 Dataset Information
The dataset contains information related to:
- Customer demographics
- Product categories
- Orders
- Purchase amount
- Occupation
- Region/Zone
- Marital status

---

# 📊 Data Cleaning Process
The following preprocessing steps were performed:
- Removed unwanted columns
- Handled missing values
- Removed duplicate records
- Converted data types
- Checked dataset information and statistics

---

# 📈 Exploratory Data Analysis
The following analyses and visualizations were performed:

1. Zone Wise Product Sales  
2. Product Category vs Orders  
3. Age Group vs Purchase Amount  
4. Gender Wise Product Orders  
5. Product Category vs Amount  
6. Zone Wise Average Purchase Amount  
7. Product Purchase Based on Marital Status  
8. Occupation Wise Purchase Amount  
9. Zone Wise Gender Count  
10. Age Distribution  
11. Age vs Purchase Amount  
12. Correlation Heatmap  

---

# 🔍 Key Insights
- Different zones show different product purchasing trends.
- Some product categories receive significantly higher orders.
- Certain age groups contribute more to overall sales.
- Male and female customers show different purchasing behavior.
- Some occupations contribute higher purchase amounts.
- Customer spending patterns vary across different regions.
- Marital status influences product purchasing preferences.
